# ClimateScope – Global Weather Analytics & Visualization

## 📌 Overview
Data analytics project analyzing 124k+ global weather records to extract climate patterns using Python and Power BI.

## 🎯 Objective
To clean, process, and analyze large-scale weather data and extract insights related to temperature, humidity, and extreme weather events.

## ⚙️ Tech Stack
- Python (Pandas, NumPy, SciPy)
- Power BI
- Jupyter Notebook

## 🧹 Data Processing
- Handled missing values and duplicates
- Removed redundant multi-unit columns
- Processed 124,000+ records

## 📊 Key Analysis
- Correlation analysis between weather variables
- Anomaly detection using 3σ method
- Feature engineering (DTR, climate zones)

## 📈 Visualization
- Power BI dashboard with geospatial analysis
- Seasonal trend comparisons
- Climate variability insights

## 📌 Key Insights
- Strong inverse correlation between temperature and humidity (-0.63)
- Identified extreme weather anomalies using statistical thresholds
- Regional climate variability patterns across hemispheres
