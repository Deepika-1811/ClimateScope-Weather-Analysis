# 🌍 ClimateScope – Global Weather Analytics & Visualization

## 📌 Overview
ClimateScope is a data analytics project that processes **124,000+ global weather records** to extract meaningful climate insights using Python and Power BI. The project focuses on data preprocessing, feature engineering, statistical analysis, and interactive visualization to understand global climate patterns.

---

## 🎯 Objective
To transform raw weather data into actionable insights by:
- Cleaning and preprocessing large-scale datasets  
- Performing statistical and exploratory data analysis  
- Detecting anomalies in weather conditions  
- Visualizing global climate trends using Power BI  

---

## ⚙️ Tech Stack
- Python (Pandas, NumPy, SciPy, Matplotlib)  
- Jupyter Notebook  
- Power BI  
- CSV-based data processing  

---

## 🧹 Data Processing Pipeline
- Processed **124,000+ weather records**  
- Handled missing values and duplicate entries  
- Removed redundant multi-unit features (Fahrenheit/Celsius)  
- Engineered key features such as:
  - Diurnal Temperature Range (DTR)  
  - Climate Zone Classification  
  - Pressure Variation Metrics  
- Converted time-series data for seasonal and regional analysis  

---

## 📊 Data Analysis
- Correlation analysis between key weather variables  
- Identified inverse correlation between temperature and humidity (-0.63)  
- Time-series analysis of seasonal climate variations  
- Regional comparison of weather patterns across hemispheres  
- Anomaly detection using 3σ (standard deviation method)  

---

## 📈 Power BI Dashboard
An interactive Power BI dashboard was developed to visualize global climate patterns:

- Temperature and humidity distribution across regions  
- Climate anomaly detection and extreme event visualization  
- Seasonal trend analysis and comparisons  
- Geospatial mapping of weather variability  
- Regional climate instability patterns  

📁 File: `ClimateScope_Dashboard.pbix`

---

## 📌 Key Insights
- Strong inverse relationship between temperature and humidity (-0.63)  
- Extreme weather anomalies detected using statistical thresholds  
- Northern Hemisphere shows higher seasonal variability  
- Certain regions exhibit consistently unstable climate behavior  
- Wind speed and pressure changes are strong indicators of precipitation events
